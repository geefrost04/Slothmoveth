## Artifacts

โฟลเดอร์นี้เก็บไฟล์ภาพที่ไม่ใช่ asset ใช้งานจริงของเว็บ

- `generated/` เก็บภาพที่ generate หรือ export ออกมาระหว่างทำงาน
- `review/` เก็บภาพ screenshot และภาพตรวจงานหน้าจอ

หมายเหตุ:
- รูปที่หน้าเว็บใช้งานจริงยังอยู่ใน `slothmove-platform/public/pic/` และ `homepage-nextjs/public/pic/`
- ห้ามย้ายรูปจาก `public/` หากยังไม่ได้เช็ค reference ในโค้ดและหน้าเว็บก่อน
